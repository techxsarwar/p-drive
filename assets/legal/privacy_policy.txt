P-Drive allows users to upload files and content.

Encryption

P-Drive is designed so that files may be encrypted before storage.

Where encryption is enabled:

File contents are encrypted prior to storage.

P-Drive personnel are not intended to access file contents during normal operation.

File contents may not be readable without the appropriate decryption credentials.

Metadata

While file contents may be encrypted, certain metadata may still be processed, including:

File names

File sizes

Upload timestamps

Download timestamps
